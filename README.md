# rick-morty-next-app
Rick and Morty app developed with Next.js and Typescript

## Deployment app 
`https://rick-morty-next-app.vercel.app/`

## Run
```
    npm install
    npm run dev
```

```
    yarn install
    yarn run dev
```
Starts on `http://localhost:3000`

## Build
```
    npm run build
```

## Development Technologies
* Next.js 
* Typescript
* GraphQL
* Jest & React testing library
